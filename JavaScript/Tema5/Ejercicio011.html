- Por seguridad los navegadores no permite escribir en una ventana ya creada.

- El primer botón abre una ventana, si está abierta el segundo botón no funciona.

- Si se cierra la primera (o no se abre) el segundo botón crea la ventana con el texto solicitado.

- Si creo una función para mover la pestaña, esta se mueve al ser creada y no cuando pulsamos el botón.
Esto puede deberse a que la variable se ejecuta antes de ser llamada por que JavaScript reordena el código.
